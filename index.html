SETUP GUIDE▶
1
Choose AI Provider & Get API Key
Select a provider from the list above. All have free tiers:
• Groq — fastest, 14k req/day. Go to console.groq.com → Sign in with Google → API Keys → Create.
• OpenRouter — smartest free models (Llama 3.3 70B). Go to openrouter.ai/keys → Create Key.
• Gemini — free, supports web search. Go to aistudio.google.com → Get API Key.
• Claude Haiku — smartest overall, paid ($0.25/1M tokens). Go to console.anthropic.com → API Keys.
2
Load Documents
Two options:
A) Upload locally — click "Choose files" and select PDF, DOCX, or PPTX from your computer. Files stay on your device.
B) Google Drive — paste a public file or folder link. File must be shared: right-click → Share → Anyone with the link → Viewer.
For folders: you also need a Google API Key (console.cloud.google.com → Enable Drive API → Create API Key).
3
Ask Questions
Type your question in the chat box. The app will:
1. Translate non-English queries to English for better search
2. Find the most relevant passages in your documents
3. Send those passages to the AI to generate an answer
4. Show the source document and page number

⚡ Tip: Use technical terms exactly as they appear in the document (e.g. "Noise Logging Tool" not "noise meter").
4
Security Notes
• Your API keys are saved only in this browser (localStorage). Never shared.
• Documents are parsed locally — they do NOT leave your device.
• Only small text excerpts (chunks) are sent to the AI provider to generate answers.
• For sensitive/confidential documents, use a private server deployment instead.
